    alternative to tee with tail

    The tail command in linux is something that you can use to observe a file in real time. Use the -f option for this to happen.

    Example, to track the output from this command:

    while true; do date >> /tmp/observation-file; sleep 1; done

    You can do..

    $ tail -f /tmp/observation-file
